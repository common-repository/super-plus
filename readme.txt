== Description ==
Super-Plus是一款Wordpress综合优化增强插件，集合多种Wordpress优化增强功能于一身,专注于改善Wordpress的易用性、稳定性与专业性。<br />

== Installation ==
1. 通过 WordPress 插件库或者手动安装 Super-Plus 并启用
2. 启用后将自动跳转至设置页面，按照需要开启对应功能保存设置即可

== Frequently Asked Questions ==
Q:Splus插件自身错误,导致博客异常<br />
A:在Splus插件更新发布之前作者都会进行调试，如果出现了问题可以反馈到博客http://www.xmgo.cc/super-plus或邮箱all-right#139.com，收到反馈后我会第一时间进行处理。

Q:Splus插件功能与当前使用主题发生冲突<br />
A:停用Splus插件中开启的与当前使用主题发生冲突的功能,请勿开启当前使用主题自带的相同的功能。如果仍有问题可以反馈到http://www.xmgo.cc/super-plus或all-right@139.com。
